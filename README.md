# Task Tracker

Platforma do zarządzania zadaniami i współpracy zespołowej.

Celem projektu jest stworzenie aplikacji webowej opartej na architekturze mikrousług, umożliwiającej zarządzanie zadaniami w zespołach projektowych.

## Główne funkcjonalności

- Rejestracja, logowanie oraz zarządzanie kontami użytkowników
- Tworzenie i zarządzanie projektami zespołowymi
- Dodawanie, edytowanie i usuwanie zadań w ramach projektów
- Przypisywanie zadań do użytkowników oraz określanie priorytetów i terminów
- Zmiana statusów zadań (np. nowe, w trakcie, zakończone)
- Śledzenie historii zmian i audyt działań użytkowników
- Powiadomienia o zmianach w zadaniach z użyciem reaktywnych strumieni
- Wyszukiwanie i filtrowanie zadań według różnych kryteriów
- Generowanie prostych statystyk i podsumowań postępu prac
- Udostępnianie funkcjonalności poprzez REST API
- Uruchamianie aplikacji w środowisku kontenerowym z wykorzystaniem Dockera
