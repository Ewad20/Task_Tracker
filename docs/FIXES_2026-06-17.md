### Issues Resolved

#### 1. PowerShell Encoding Errors

**Problem:** 
The `build.ps1` script contained UTF-8 special characters (Polish diacritics, em-dashes, Unicode symbols) that caused PowerShell parsing errors on Windows systems.

**Error Messages:**
- "The string is missing the terminator"
- "Unexpected token"
- "FormatException" in ToString() method

**Solution:**
Replaced all non-ASCII characters with ASCII equivalents:
- Polish characters (ą, ć, ę, ł, ń, ó, ś, ź, ż) → ASCII equivalents (a, c, e, l, n, o, s, z, z)
- Em-dash (—) → Hyphen (-)
- Unicode box-drawing characters (─) → Hyphens (-)
- Checkmark (✓) → [OK]
- Cross mark (✗) → [ERROR]
- Fixed date format string: `ToString('mm:ss.ff')` → `"{0:mm\:ss\.ff}" -f $elapsed`

**Files Changed:**
- `Task_Tracker/build.ps1`

**Verification:**
